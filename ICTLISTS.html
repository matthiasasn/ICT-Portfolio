LESSON 2: TYPES OF LISTS


    Unordered List
  • Only has one attribute: type
  • type attribute has 3 values: disc, circle, and square
  • disc is used for round bullets
  • circle is used for circular bullets, which is used in this unordered list!
  • square is used for square bullets
  • Unordered lists have solid round bullets by default if it doesn't use the type attribute.
    Ordered List & Nested List
  1. Ordered list has 2 attributes: type and start.
  2. type attribute has 5 values: 1, A, a, I, and i
    • 1 is used for regular numbering, and is also the default type for ordered lists.
    • a for lowercase alphabet
    • A for uppercase alphabet
    • i for lowercase Roman Numerals
    • I for uppercase Roman Numerals, which is used in this ordered list!
  3. start attribute is used when you want to start with a different number or alphabet.
  4. The title is "Ordered List & Nested List" since I also included an unordered list here! You can nest lists by putting ordered or unordered lists inside the li tag.
